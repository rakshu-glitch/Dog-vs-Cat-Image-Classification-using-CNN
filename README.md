# Dog vs Cat Image Classification using CNN (Transfer Learning)

## Project Description

This project classifies images of cats and dogs using Deep Learning.

The model is built using TensorFlow and MobileNetV2 Transfer Learning.

Dataset:
Microsoft Cats and Dogs Dataset (PetImages)

## Technologies Used

- Python
- TensorFlow
- Keras
- MobileNetV2
- NumPy
- Matplotlib

## Project Structure

train.py
predict.py
clean_dataset.py
model.keras
accuracy_graph.png
PetImages/

## How to Run

Install dependencies

pip install -r requirements.txt

Train model

python train.py

Predict image

python predict.py

## Output

Prediction: Dog

Confidence: 99.42%
